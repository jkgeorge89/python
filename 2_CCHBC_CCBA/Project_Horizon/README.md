# Project Horizon

This directory is organized by workstream so the different valuation iterations do not sit on top of each other.

The `game_theory/` folder is preserved as the original binomial game-theory approach. Any exploratory successor method sits outside that folder in its own method-specific workstream.

## Folder structure

- `proxy/` — original proxy model and proxy-era supporting notes
- `full/` — linked-contract FULL valuation notebook and memo
- `game_theory/` — original binomial strategic game-theory methodology, notebook, and chart specification
- `high_resolution_tree/` — exploratory higher-resolution strategic tree work kept separate from the original binomial game-theory work
- `continuous_state_qvi/` — daily continuous-state dynamic-programming reference solver kept separate from the tree-based workstreams
- `references/` — agreement extracts, key-terms summary, and source documents
